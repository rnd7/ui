Methods

<static> bind(str, callback)

Binds a key combination to a callback function. Supports unicode characters as well as modifier key combinations.
Parameters:
Name Type Description
str string A valid modifier character combination
callback function The callback method that shall be invoked if the key combination is recognized
Example
keyboard.bind(
  "SHIFT+ALT+a",
  function(ev) {
    console.log(ev)
  }
);

<static> unbind(str)

Unbind a key combination
Parameters:
Name Type Description
str string The String used to bind a combination
Example
unbind("SHIFT+ALT+x");
